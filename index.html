upside

It provides advanced autocompletion, navigation, and refactoring. Having such tools is almost a requirement for large projects.

TypeScript offers the best developer tooling experience available in the JavaScript world today.

Downside

Type correctness does not guarantee program correctness.

The gist: Static types don’t reduce over-all bug density by much, but you may still love them for the developer tooling.

Type annotations obviously create more syntax noise, and that syntax makes code harder to read, and harder to maintain.
